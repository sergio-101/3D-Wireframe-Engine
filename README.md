# A 3D Wireframe Engine

A first-person 3D wireframe engine built from scratch in vanilla JavaScript using the Canvas 2D Context APIs. You don't need WebGL and Three.js for every little thing.

## Controls

- W / S — Move forward / back
- A / D — Strafe left / right
- Shift — Sprint
- Mouse — Look (click canvas to lock)
